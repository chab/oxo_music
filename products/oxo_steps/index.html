Oxo Steps

MIDI step sequencer / arpeggiator

Change chord, everything follows

The sequencer we couldn't find: enter one chord, shape six independent sequences.

Example

  • Track 1 (Channel 1): Bass line from the chord's lowest note.
  • Track 2 (Channel 2): Chord stabs from selected chord notes, or fixed notes if preferred.
  • Track 3 (Channel 3): Arpeggio using all notes, polymetric + 1 octave every 7 steps
  • Track 4 (Channel 2): Random in-key notes every now and then (Note the channel : it is the same instrument playing the chord).

Change chord, everything follows.

Build scenes for live flow

Build up to 8 scenes. Each scene stores a chord plus pattern changes per track, so you can move through ideas quickly while staying musical.

Capture and continue in your DAW

Record your performance as a MIDI file and continue the idea directly in your DAW with the same harmonic and rhythmic structure.

Feature highlights

  • Polymetric lanes (e.g. notes over 8, octave over 3)
  • Pattern direction: forward, backward, ping-pong, random
  • Predictable probability lane controls
  • Chord memory
  • Cross-pattern hold/legato
